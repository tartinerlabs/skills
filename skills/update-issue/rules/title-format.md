**Rule**: Use natural, descriptive language. Match the style of existing issue titles in the repository.

### Incorrect

```
fix: update login flow
feat: improve search
```

### Correct

```
Update login flow to handle expired sessions
Improve search with fuzzy matching
```

### Guidelines

- Natural language, no conventional commit prefixes
- Keep concise but informative
- Match existing title style in the repository
