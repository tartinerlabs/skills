# TypeScript

Type checking with strict mode enabled.

## Install

```bash
<pm> add -D typescript
```

## Config

Create `tsconfig.json`:

```json
{
  "compilerOptions": {
    "strict": true,
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "esModuleInterop": true,
    "skipLibCheck": true,
    "resolveJsonModule": true,
    "isolatedModules": true
  }
}
```

## Framework Detection

If a framework is detected, extend from its recommended tsconfig instead of creating from scratch:

| Framework | Extend from |
|-----------|-------------|
| Next.js | `next/tsconfig` (auto-generated by Next.js) |
| Vite | `@tsconfig/node22/tsconfig.json` |
| Remix | `@remix-run/dev/tsconfig` |
| Expo | `expo/tsconfig.base` |

When a framework is detected, skip creating `tsconfig.json` and inform the user to use the framework's setup command instead.
