title: Issue Title Format
impact: HIGH
tags: issue, title, natural-language
---

**Rule**: Use natural, descriptive language. No conventional commit prefixes.

### Incorrect

```
feat: add dark mode support
fix: broken login redirect
chore: update dependencies
```

### Correct

```
Add dark mode support
Login redirect fails after token expiry
Update React and TypeScript dependencies
```

### Guidelines

- Clear and specific to the problem or feature
- Concise but informative
- Match the style of existing issue titles in the repository
